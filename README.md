# CodeAlpha - Basic Network Sniffer

## Project Overview

This project was developed as part of the CodeAlpha Cyber Security Internship.

The Basic Network Sniffer captures live network packets and displays useful information such as source IP address, destination IP address, protocol type, and packet details. It helps in understanding network communication and packet structures.

## Features

- Captures live network traffic
- Displays Source IP Address
- Displays Destination IP Address
- Identifies TCP, UDP, and ICMP protocols
- Displays packet summaries
- Helps understand network packet flow

## Technologies Used

- Python
- Scapy

## Installation

Install Scapy using:

pip install scapy

## Running the Program

Run the following command:

python network_sniffer.py
